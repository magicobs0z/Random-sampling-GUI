# Random-sampling-GUI
这是一个拥有用户界面的随机抽取工具，有许多可调选项
## 介绍
代码的第9行放的就是你想要的文字,把这里替换成你想要的随机字符串，然后编译就可以了
注释写得很清楚了，各种功能都可以加，改，你可以随便使用这个项目，只要不删除第1,2行注释就可以
特别简单的一个项目，希望可以给你的生活带来方便
